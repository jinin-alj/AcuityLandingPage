# Assets Guide

## Background images (required for design)
Browsers cannot use PDFs as background images. Export the first page of each PDF to PNG:

1. **Acuity base** → `assets/acuity-base.png` (sharp background, shown on load)
2. **Blurry Acuity** → `assets/blurry-acuity.png` (blurred, shown when scrolling)

## Logo
Export **White background logo.pdf** to PNG or SVG → `assets/logo.png` or `assets/logo.svg`
